Weather App 🌦️

A responsive weather application built using HTML, CSS, and JavaScript that fetches live weather data using the OpenWeather API.

🚀 Features
Search weather by city name
Live temperature display
Humidity and wind speed
Dynamic weather icons
Error handling for invalid city names
Responsive modern UI


🛠️ Technologies Used
HTML
CSS
JavaScript
OpenWeather API
